# 🎮 Mario Jump

🎮 **Mario Jump** é um mini-jogo feito com HTML, CSS e JavaScript onde o jogador controla o Mario para desviar de obstáculos.  
Um projeto simples, nostálgico e divertido para praticar lógica e animações na web!
